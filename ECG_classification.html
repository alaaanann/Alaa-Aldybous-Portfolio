ECG Heart Abnormality Classification

MATLAB Machine Learning Signal Processing Deep Learning CNN Mathematical Modeling Classification

Status: Unpublished Manuscript/In Progress


Role: Lead Researcher/First Author

Project Overview

This project combines signal processing and machine learning to explore the classification of heart abnormalities using Electrocardiogram (ECG) signals. The primary objective was to build and train a machine learning model capable of classifying ECG signals as either 'normal' or 'abnormal'.

The project focused on mathematically modeling three specific abnormal conditions: Premature Ventricular Contraction (PVC), Atrial Fibrillation (AFib), and Ventricular Tachycardia (VTach). A custom dataset was constructed by generating synthetic signals for these conditions and combining them with experimental data from healthy individuals. This dataset was then used to train a classifier, which was ultimately tested on new, unseen ECG signals to predict for abnormalities.

Key Contributions & Skills

As the lead researcher, I was responsible for all technical stages of the project, from modeling to final prediction.

  • - Signal Processing & Mathematical Modeling: Researched and mathematically modeled a normal ECG signal by defining its core components: the P-wave, QRS complex, and T-wave. Developed distinct mathematical models to simulate the specific waveform changes associated with PVC (omitted P-wave, wide QRS), AFib (irregular R-R intervals, absent P-wave), and VTach (rapid, wide QRS complexes with no P or T-waves).
  • Synthetic Data Generation & Curation: Wrote MATLAB scripts to generate a synthetic dataset of abnormal ECG signals based on the defined mathematical models. Integrated experimental ECG data from 10 healthy individuals with the generated abnormal signals to create a labeled dataset for training.
  • Machine Learning (CNN): Processed and prepared the final dataset, splitting it into training and testing sets to validate model performance. Implemented and trained a Convolutional Neural Network (CNN) using MATLAB's Deep Learning Toolbox to learn the complex, time-series features distinguishing normal signals from the three types of abnormalities. Evaluated the trained model's performance on the unseen testing set. The model demonstrated a clear ability to learn from the data, with classification accuracy significantly exceeding the 25% random-chance baseline for this 4-class problem.
  • Predictive Modeling: Applied the final trained model to a new dataset from 10 new individuals to predict and classify their heart status as healthy or abnormal.

Tools & Technologies

MATLAB: Utilized for all project phases, including:

  • - Signal Processing Toolbox: For modeling, generating, and preparing the ECG waveforms.
  • - Deep Learning Toolbox: For designing, building, training, and testing the Convolutional Neural Network (CNN).
  • - Core MATLAB functions for data management and analysis.
