I-V Performance Analysis of Solar Cells Using Numerical Method

2023 C++ Numerical Methods Python Flask pybind11

The Challenge

Accurately predicting the power output of photovoltaic (PV) systems is critical for their design and optimization. The performance of a solar panel, described by its I-V (Current-Voltage) and P-V (Power-Voltage) characteristic curves, changes significantly with environmental conditions like solar irradiance and temperature. The challenge was to develop a tool that could precisely calculate and visualize this performance using real-world data and robust numerical methods.

Our Solution

We developed a web application using a C++ backend, a Python Flask web framework, and pybind11 to bridge the two. The application determines the power output of an MSX-60 solar panel by prompting the user to enter a specific time. Using data from a CSV file and the single-diode mathematical model, the program calculates the panel's I-V and P-V curves using the Newton-Raphson method. The tool also allows users to input new temperature or irradiance values to compare performance curves and understand their impact.

Methodology

The application's methodology is divided into several key functions that handle data processing, numerical calculation, and user interaction.

1. Data Handling and Interpolation (TempIrraVal function)

  • - The application first reads environmental data (time, temperature, irradiance) from a CSV file into a 2D vector.
  • - When a user inputs a time, the system checks for an exact match in the data.
  • - If no exact match is found, linear interpolation is used to accurately estimate the temperature and irradiance values for the user's specified time based on the nearest data points.

2. Core Calculation Engine (MaxVoltage and Max_power functions)

  • - The core of the application utilizes the Newton-Raphson method to solve the complex, non-linear single-diode model equation for the solar cell.
  • - First, it iteratively calculates the maximum voltage (V_oc) where the current is zero.
  • - Then, for a range of voltage steps from 0 to the maximum, it uses the Newton-Raphson method again to calculate the corresponding current for each step, generating the full I-V curve. The P-V curve is then derived from these values.

3. Interactive Analysis (SetIrra and SetTemp functions)

  • - The application allows users to go beyond the initial dataset by inputting a new temperature or a new irradiance value.
  • - The system then recalculates the I-V and P-V curves based on this new input and plots them on the same graph as the original curves. This provides a direct visual comparison, making it an excellent interactive educational tool.

Conclusion & Impact

This project successfully created a program to aid in the design and optimization of PV systems. By implementing the Newton-Raphson method in C++ and integrating it with a user-friendly web interface, the tool provides accurate calculations and clear visualizations of a solar panel's performance. This work contributes to the pursuit of sustainable energy by providing a practical tool for understanding and improving the efficiency of photovoltaic technology.
